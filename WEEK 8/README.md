# WEEK 8: RESPONSIVE WEB DESIGN WITH MEDIA QUERIES

---

## Overview

Week 8 focuses on building responsive web pages using a mobile-first approach, flexible layouts, and CSS media queries. The work for this segment includes:

1. A responsive personal profile page
2. A responsive product showcase page

Both tasks were implemented in HTML and CSS and tested across different screen sizes to confirm that the layouts adapt correctly on mobile, tablet, and desktop screens.

---

## Task 1: Responsive Profile Page

**Files:** `profile.html`, `task1_profile.html`

This task demonstrates how a profile page can be structured to respond to different viewport sizes. The design includes a navigation bar, profile header, and content panels for About, Skills, and Contact information.

### Key Responsive Features

- Mobile-first layout using a single-column structure
- Media queries for tablet and desktop breakpoints
- Flexible header layout that changes from stacked to horizontal
- Content panels that expand from one column to two columns and then three columns
- Responsive image sizing and readable typography

## Task 2: Responsive Product Showcase

**Files:** `task2_products.html`, `showcase.html`

This task applies responsive CSS Grid principles to a product listing page. The page displays product cards with images, names, descriptions, prices, and interface controls such as category tabs and cart access.

### Key Responsive Features

- Mobile-first product grid
- Grid changes from one column on small screens to multiple columns on larger screens
- Responsive product images using `aspect-ratio` and `object-fit`
- Flexible spacing and typography across breakpoints
- Navigation and filter controls that adapt for wider screens
